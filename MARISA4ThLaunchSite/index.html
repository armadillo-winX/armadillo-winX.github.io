MARISA for 東方管制塔

Multi Advanced Replay files Import and Sort Add-on for 東方管制塔

これは何?
Multi Advanced Replay files Import and Sort Add-on for 東方管制塔、通称 MARISA for 東方管制塔 は、 東方管制塔 NX にリプレイファイル管理機能を追加するための東方管制塔向けプラグインです。
各作品のリプレイフォルダにリプレイファイルを追加したり、各作品のリプレイフォルダからリプレイファイルの削除やリネームを行えます。
